# Лабораторные работы по фреймворкам ИИ 

В репозитории представлены результаты **5 лабораторных работ** в формате **Jupyter Notebook**.


## Структура репозитория

- `lab1_knn.ipynb` — **ЛР-1:** KNN (классификация + регрессия)
- `lab2_logreg_linreg.ipynb` — **ЛР-2:** логистическая регрессия (классификация) и линейная регрессия (регрессия)
- `lab3_decision_tree.ipynb` — **ЛР-3:** решающее дерево (классификация + регрессия)
- `lab4_random_forest.ipynb` — **ЛР-4:** случайный лес (классификация + регрессия)
- `lab5_.ipynb` — **ЛР-5:** градиентный бустинг (классификация + регрессия) + **итоговое сравнение ЛР 1–5**

## Датасеты (Kaggle)

Во всех работах используются 2 табличных датасета с Kaggle:

1) **Классификация — Faulty Steel Plates**
- Kaggle handle: `uciml/faulty-steel-plates`
- Задача: предсказать тип дефекта (7 классов)

2) **Регрессия — Concrete Compressive Strength**
- Kaggle handle: `elikplim/concrete-compressive-strength-data-set`
- Задача: предсказать прочность бетона на сжатие 



## Метрики

- Классификация: **Accuracy**, **Macro F1**
- Регрессия: **MAE**, **RMSE**, **R²**
